DESCRIPTION

t.sentinel.import is a GRASS GIS addon Python script to download and import the Sentinel-2 scenes and create a STRDS.

EXAMPLE

t.sentinel.import s2names=s2names2.txt settings=/mnt/pgpass/.sentinel.txt \
  nprocs=7 memory=8000 -c pattern='B0(4|8)_10m' strds_output=test
